## 1. 소개
뷰아는 “뷰티”와 “아이디어”의 합성어입니다. 쇼핑 및 뷰티 관련 정보를 서로 공유하며 소통할 수 있고, 카테고리별마다 메이크업 Tip을 얻어갈 수 있습니다.

- 구글, 카카오 로그인 가능
- 쇼핑, 마이페이지, 장바구니 기능
- 뷰티 Tip 페이지
- 북마크 기능
- 커뮤니티 게시글 작성 및 댓글 기능
